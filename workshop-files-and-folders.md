# Typische Workshopordner

## Ordner- und Dateistruktur für HTML5 Workshops

```
workshop
  |- .vscode
  |     launch.json
  |     settings.json
  |- assets
  |    |- css
  |    |    style.css
  |    |- figures
  |    |    shou.svg
  |    |    landscape.png
  |    |- js
  |    |    script.js
  |    |- data
  |         user.json
  |         user-list.csv
  |
  |- docs
  |    html-basics.pdf
  |    cascading-stylesheet-basics.pdf
  |    javascript-basics.pdf
  |
  |  index.html
  |  login.php
  |
  |  .prettierrc
  |  .gitignore
```
